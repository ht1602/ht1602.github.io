Stochastic Process

A stochastic process on probability space $(\Omega,\cal{F},\Bbb{P} )$ is a collection of random variables ${X_t:t\in\cal{T}}$.

$\cal{T}$ is always countable, $\cal{T}=\Bbb{N}$.

A Markov chain is an infinite sequence of random elements $(X_t){t=1}^{\infin}$ where the conditional distribution of $X{t+1}$ given $X_1,…,X_t$ is the same as the conditional distribution of $X_t$. The sequence has the property that given the last element, the history is irrelevant to â€˜predictâ€™ the future.

Probability kernel(Markov kernel)

from $(\cal{X},F)$ to $(\cal{Y,G})$ is a function $K:\cal{X}\times G\rightarrow [0,1]$ such that

(a) $K(x,\cdot)$ is a measure for all $x\in \cal{X}$

(b) $K(\cdot, A)$ is F-measurable for all $A\in\cal{G}$

product kernel $K_1\otimes K_2$ is the probability kernel from $(\cal{X},F)\rightarrow(\cal{Y}\times Z, G\times H)$

When P is a measure on $(\cal{X},F)$ and K is a kernel from $\cal{X}$ to $\cal{Y}$, then $P\otimes K$ is a measure on $(\cal{X}\times Y,F\times G)$.

When P is a probability measure on $\cal{X}$ and $K_1$ is a kernel from $\cal{X}$ to $\cal{Y}$ and $K_2$ a kernel from $\cal{X}\times Y$ to $\cal{Z}$, then $P\otimes K_1\otimes K_2$ is a probability measure on $\cal{X}\times Y\times Z$.

Martingales and Stopping Times

A F-adapted sequence of random variables $(X_t){t\in\mathbb{N}+}$ is a $F$-adapted martingale if

(a) $\mathbb{E}[X_t|\mathcal{F}{t-1}]=\mathrm X{t-1}$almost surely for all $t\in{2,3,…}$

(b) $X_t$ is integrable

Martingales: the total winnings over time if winning a dollar for each heads and losing a dollar for each tails. The winning after t rounds is $S_t=\sum_{s=1}^tY_s$.
